Project Description
Developed for a collaborative writing app built to streamline the creation and approval of content destined for the web, this logo is a clever and unexpected combination of a pencil and a honey bee that gives a unique personality to this Cleveland-based startup. The web application features a public home page and a comprhensive web application that required meticulous prototypes and user testing to ensure high usability and adoption.
